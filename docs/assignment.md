# 온보딩 프로젝트

## 📅 진행 기간
- **2026. 03. 19. ~ 2026. 03. 23.**

## 📋 주제 및 요구사항
문의가 들어오면 자동으로 저장 및 알림까지 이어지는 **간단한 Q&A 알림 서비스**를 구현해주세요.

### 🌐 Web Application

- 문의 입력 페이지
  - 이름 / 이메일 / 제목 / 본문 입력 가능
  - 등록 버튼 제공
  - 등록 결과를 사용자에게 표시
  - 등록 버튼 클릭 시 n8n 워크플로우 호출

- 관리자 페이지
  - 문의 목록 조회 가능
  - 각 문의의 상태 확인 가능
  - 등록됨 / 처리중 / 완료됨
  - 문의 상태 변경 가능
  - 상태가 완료됨 으로 변경되면 n8n 워크플로우 호출

### ⚙️ n8n

- 문의 등록 워크플로우
  - 문의 내용을 Notion DB에 저장
  - 관리자에게 메일 전송
  - 이름 + 제목이 같으면 중복 문의로 판단
  - 중복 문의는 등록되지 않아야 함

- 완료 처리 워크플로우
  - 문의자에게 결과 메일 전송
  - 관리자에게 완료 알림 메일 전송
  - Notion DB의 상태 및 처리 결과 업데이트

> **📝 Note**
> 기술 스택과 서비스 구조는 요구사항을 만족하는 선에서 자유롭게 설계 및 활용해주세요.
